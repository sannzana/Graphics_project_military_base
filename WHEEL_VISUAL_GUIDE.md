## Material Properties Used

```
All components use Phong Material Model:

AMBIENT = 0.28 × DIFFUSE COLOR
DIFFUSE = Base Color (see table above)
SPECULAR = (0.6, 0.6, 0.6) - Light gray
SHININESS = Component-specific (8-20)

Shininess Values (for metallic effect):
?? Tire: 8 (dull rubber)
?? Outer Rim: 20 (bright metal)
?? Inner Rim: 16 (medium metal)
?? Spokes: 14 (medium metal)
?? Center Dark: 12 (less reflective)
?? Hub Outer: 18 (shiny metal)
?? Hub Shadow: 10 (dull)
?? Hub Cap: 16 (medium reflective)
```
